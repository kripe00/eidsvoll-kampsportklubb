# Eidsvoll Kampsportklubb

Dette er nettsiden for Eidsvoll Kampsportklubb. Siden brukes til å informere om klubbens tilbud, treningstider, nyheter, og for håndtering av medlemskap. 

## 🛠 Teknologi

- **Frontend:** Next.js (App Router), React, Tailwind CSS
- **CMS:** TinaCMS
- **Backend/Hosting:** Firebase
- **UI Komponenter:** shadcn/ui, Lucide React, Base UI
